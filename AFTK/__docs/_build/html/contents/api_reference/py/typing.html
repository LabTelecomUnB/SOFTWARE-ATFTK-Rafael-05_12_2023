Quantity Like Types#

The following aliases represent types that can be coerced into (can be understood as) quantities. In short, those types are numbers (scalars), numerical data (arrays, list, etc) or even quantities itself.

Note

The aliases _ArrayLike<X>_co used in the following definitions are imported from numpy._typing._array_like and denote array-like objects that can be coerced into ndarrays with <X> dtype.

Generic Physical Type#

Length#

smet.typing.LengthLike: TypeAlias = _ArrayLikeNumber_co | Quantity['length']#

Alias representing types that can be coerced into length quantities.

smet.typing.RealLengthLike: TypeAlias = _ArrayLikeFloat_co | Quantity['length']#

Alias representing types that can be coerced into real-valued length quantities.

Angle#

smet.typing.AngleLike: TypeAlias = _ArrayLikeNumber_co | Quantity['angle']#

Alias representing types that can be coerced into angle quantities.

smet.typing.RealAngleLike: TypeAlias = _ArrayLikeFloat_co | Quantity['angle']#

Alias representing types that can be coerced into real-valued angle quantities.
