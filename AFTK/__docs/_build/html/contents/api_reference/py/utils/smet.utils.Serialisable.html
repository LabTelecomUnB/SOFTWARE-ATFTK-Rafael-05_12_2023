Serialisable#

Typehints#

Serialisable.SubType = TypeVar('SubType', bound=Serialisable)#

Typehint representing instances of Serialisable subclasses.

Serialisable.PrimitiveSType TypeAlias = Union[None, str, Number, ndarray, Quantity, Timestamp, DatetimeIndex]#

Alias representing the primitive types whose instances are allowed as Attributes in Serialisable subclasses.

Serialisable.SType TypeAlias = PrimitiveStype | SubType | list[SType] | dict[str, SType]#

Serialisable Type: alias representing all types whose instances are allowed as serialisable attributes in Serialisable subclasses.

Serialisable.RType TypeAlias = PrimitiveStype | SubType | list[SType] | dict[str, SType]#

Retrievable Type:Alias representing all types whose instances can be unserialised into in Serialisable subclasses.
