Math tools (aftk.mathtools)#

This package is a collection of useful routines and high-level services involving mathematical calculations and data visualisation that supports AFTK ecossystem.

PSD Matrices (aftk.mathtools.psdm)#

This module provides performance-optimised services regarding complex-valued positive-semidefinite matrices \(A^H A\) when \(A\in\mathbb{C}^{m\times n}\) is given.

In several situations one needs to analyse some mathematical properties of square matrices in the form \(A^H A \in\mathbb{C}^{n\times n}\) where \(A\in\mathbb{C}^{m\times n}\) is already known. In scenarios like those, the previous knowledge of \(A\) can be used to increase the performance of the required calculations.

Recommended import#

from aftk.mathtools import psdm

Implemented Services#

psdm.eigenvalues

Calculates the eigenvalues of the PSD matrix \(M = A^H A\).

psdm.condition_number

Calculates the condition number of the PSD matrix \(M = A^H A\).

psdm.inverse

Calculates the inverse of the PD matrix \(M = A^H A\).

Spherical Harmonics (aftk.mathtools.spharm)#

Antenna fields are most commonly described in spherical coordinates where they are usually projected in orthogonal set of functions involving spherical harmonics. In this context, this module provides services to quickly calculate those functions.

Convention: \(\theta \in \left[0, \pi\right]\) and \(\phi \in \left[0, 2\pi\right)\) are the polar and azimuthal angle, respectively. I

Recommended import#

from aftk.mathtools import spharm

Implemented Services#

spharm.Ylm

Calculates the spherical harmonic \(Y_{\ell}^m\left(\theta, \phi\right)\).

spharm.m_times_Ylm_over_sin_theta

Calculates \(\dfrac{mY_{\ell}^m\left(\theta, \phi\right)}{\sin\theta}\).

spharm.dYlm_dtheta

Calculates \(\dfrac{\partial Y_{\ell}^m}{\partial\theta}\left(\theta,\phi\right)\).

spharm.Tlm

Computes the matrix-eigenfunction \(\mathbf{T}_{\ell m}\left(\theta, \phi\right)\).

spharm.T

Computes the model matrix \(\mathbf{\mathsf{T}}\left(\theta, \phi\right)\).

spharm.Upsilon

Computes the regressor matrix \(\mathbf{\Upsilon}\left(\theta, \phi\right)\).
