spharm.dYlm_dtheta#

aftk.mathtools.spharm.dYlm_dtheta(int l, int m, double theta, double phi) → double complex#

Calculates \(\dfrac{\partial Y_{\ell}^m}{\partial\theta}\left(\theta,\phi\right)\).

Parameters:
lint

degree. Must be greater than zero.

m: int

order.

theta: double

polar angle in radians

phi: double

azimuthal angle in radians

Returns:
outcomplex

The \(\ell\)-degree \(m\)-order second spherical eigenfunction \(\dfrac{\partial Y_{\ell}^m}{\partial\theta}\left(\theta,\phi\right)\)
